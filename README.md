# CreatorFlow AI

## Features
- AI content generation
- Viral scoring
- Content calendar
- Analytics dashboard
- Role based access control

## Roles

### Viewer
- View content
- View analytics

### Editor
- Create content
- Save content
- Manage own content

### Admin
- Manage users
- Change roles
- View all content

## Tech Stack
- Next.js
- Supabase
- OpenAI
- Vercel
